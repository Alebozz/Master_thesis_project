Thanks in advance for your participation, here below are the instructions of how the program works:

In this survey you will be presented 20 random samples out of 88 possible samples. 

For each sample you will see 2/3 feature discriminancy maps images computed using Fisher's score method over PSDs of single runs 
for that subject plus an additional feature discriminancy map image that is obtained from the concatenation of the single runs

The feature map that represent the concatenation of the single runs has an interactive grid where each cell is clickable
Every cell of the grid correspond to a given pair Channel-Frequency feature for the discrimination of the Both Hands vs. Both Feet motor imagery

Your task is to select the cells of the grid that you consider to be the most useful for training a classifier that will
be used for distinguishing the two classes.
The application presents you several buttons that can be used to display additional images that can be useful to guide the decision
about the cells to select.

In particular it will be possible to see the ERD/ERS spectrograms of each individual channel of the 15 available by simply selecting the corresponding
cells on the 5x3 grid that has the labels of the channels and then clicking the button below the grid to open the windows that will display the plots.
Other than that, you will be able to have a look at the ERD/ERS values over time under the form of topoplots in the mu(8-14Hz) and beta(14-30Hz) band

In order to confirm the selection of the cells for the given sample you will need to press the enter key.
After the enter key is pressed, the choices are saved automatically to a .txt file that will be located in the same folder of the python script you are executing
Once you confirm your choices, new sample will be loaded and displayed in order to be evaluated

After the last sample of the 20 presented is evaluated, the program will close automatically.
IMPORTANT: the program can be manually closed before all the 20 samples have been analyzed since every time a sample is analyzed and the enter key is pressed,
the program automatically saves the choices for that sample.

The survey can be taken as many time as ones wants since all the choices will be stored in the same .txt file without deleting previous results.

For any other information, please refer to the GitHub page where you downloaded this program.
